About This Website

Project Overview

This is a fully functional Job Portal Website built entirely by me as a fresher. It allows companies to post jobs and users to register, apply for jobs, and manage their applications through a secure dashboard.

Technologies Used

  • Frontend: HTML5, CSS3, Bootstrap 5, EJS Templating
  • Backend: Node.js, Express.js
  • Authentication: Passport.js (including Google OAuth)
  • Database: PostgreSQL with pgAdmin
  • Mailing: Nodemailer (for email communication)
  • Other: Multer for file uploads, express-session, dotenv, connect-flash

My Role & Skills

I designed and developed this entire website from scratch, including:

  • Creating responsive and clean user interfaces
  • Designing backend APIs for job management
  • Implementing secure user authentication
  • Building resume upload and job application workflows
  • Using PostgreSQL for efficient data storage and queries
  • Sending professional emails to job posters using Nodemailer
  • Managing user sessions and dashboard routing

This project showcases my ability to build full-stack web applications with a professional approach and clean code architecture.
